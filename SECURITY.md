# Security Policy

InvestLion takes security seriously and is committed to responsible disclosure.

---

## Reporting a Vulnerability

If you discover a potential security issue in the InvestLion smart contract,
documentation, or related infrastructure, please report it privately.

Do NOT disclose vulnerabilities publicly until they have been reviewed and addressed.

### Contact
- Telegram: @elzose
- Project: InvestLion (ILION)

---

## Scope

This security policy applies to:
- ILION smart contract
- Official documentation in this repository
- Any future InvestLion-related infrastructure

---

## Safe Harbor

We encourage responsible disclosure.
Researchers acting in good faith will not be penalized for reporting vulnerabilities responsibly.

---

## Disclaimer

No funds should be sent to any address claiming to represent InvestLion unless officially announced through verified channels.
